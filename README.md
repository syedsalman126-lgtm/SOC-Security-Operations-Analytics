# SOC Security Operations Analytics using PostgreSQL

This project simulates real-world Security Operations Center (SOC) workflows
using PostgreSQL. It focuses on monitoring, detection, investigation, and
executive-level reporting.

## Tools
- PostgreSQL
- SQL
- VS Code
- Git & GitHub

## SOC Coverage
- L1 Monitoring & Alert Triage
- L2 Detection & Investigation
- L3 Reporting & Threat Intelligence
